Add team

By default, SkilledTree will create ‘default’ team to your company. You can add many more team in SkilledTree by manually creating the team in the ‘Settings’ page.

Edit team

In case, if you want to edit/delete certain team, you can update it in the ‘Settings’ page. Kindly note that you can delete the team only if the team doesn’t have any employees and recruiters.

Add recruitment team to the portal

Once the team have been created, you can add recruiters to handle each subnetwork. The recruiters will receive an email with the credentials to log into the portal and manage their team.

Resend credentials to HRs

Resend the credentials via Email / SMS in case the recruiters did not receive the email.

Give and revoke admin access to recruiters

With admin access, a recruiter can unlock the functionality and it is available to the ‘Admin’ of the company. For instance, only the admin has the authority to add recruiters to the team. If any other recruiter have an admin access, he can add recruiters to the company network too.
